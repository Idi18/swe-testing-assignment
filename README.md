# Quick-Calc

Quick-Calc is a simple calculator application created for the Advanced Software Engineering assignment on Software Testing & Quality Assurance.  
It supports basic arithmetic operations and a clear function, with an emphasis on clean code, unit testing, and integration testing.

## Features

- Addition
- Subtraction
- Multiplication
- Division (handles division by zero gracefully)
- Clear (resets result to 0)

## Setup

### Requirements
- Python 3.x

### Install dependencies

```bash
python -m pip install pytest

```markdown
